# ⚡ Executor Agent

## Owner

**Member 2**

## Goal

Execute the deployment plan generated by the Planner Agent. The Executor Agent is the second agent in the pipeline. It receives the structured plan (JSON) from the Planner, calls the AWS infrastructure module to create an EC2 instance, then calls the deployment module to SSH into the instance and deploy the Flask application. It returns the final deployment result with the public IP and URL.

## Input

Structured plan from the Planner Agent:
```json
{
  "app": "flask",
  "instance_type": "t2.micro",
  "region": "ap-south-1"
}
```

## Output

Deployment result:
```json
{
  "status": "success",
  "ip": "54.xxx.xxx.xxx",
  "url": "http://54.xxx.xxx.xxx:5000"
}
```

## Flow

```
Plan (from Planner)
    ↓
1. Call infra/aws/ec2.py → create_instance(plan)
    ↓
2. Get instance_id + public_ip
    ↓
3. Call infra/deploy/flask_deployer.py → deploy(ip)
    ↓
4. Return { status, ip, url }
```

## File

| File           | Purpose                                          |
|----------------|--------------------------------------------------|
| `executor.py`  | ExecutorAgent class with `run(plan)` method      |

## Dependencies

The Executor Agent does NOT implement AWS or deployment logic directly. It calls:

| Module                          | Function          | Purpose               |
|---------------------------------|-------------------|-----------------------|
| `infra.aws.ec2`                 | `create_instance()` | Create EC2 instance  |
| `infra.deploy.flask_deployer`   | `deploy()`          | Deploy Flask via SSH |

## Example Logic

```python
# executor.py (pseudocode)
from infra.aws.ec2 import create_instance
from infra.deploy.flask_deployer import deploy_flask

class ExecutorAgent:
    def run(self, plan: dict) -> dict:
        # Step 1: Create EC2 instance
        instance = create_instance(plan)
        ip = instance["ip"]

        # Step 2: Deploy Flask app
        deploy_flask(ip)

        # Step 3: Return result
        return {
            "status": "success",
            "ip": ip,
            "url": f"http://{ip}:5000"
        }
```
